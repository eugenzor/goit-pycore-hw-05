# goit-pycore-hw-05

Тема 8. Домашня робота. Функціональне програмування та вбудовані модулі Python


## Запуск

Ініціалізуція віртуального середовища
```bash
source .venv/bin/activate
```

Запуск всього разом
```bash
python main.py
```

Запуск кожного завдання окремо
```bash
python task1.py
python task2.py
python task3.py <шлях_до_лог_файлу> [рівень_логування]
python task4.py
```
